Projects

1. Breast Cancer Classification System using Machine Learning

  • Engineered a machine learning model to classify breast cancer as benign or malignant.
  • Implemented using Logistic Regression, KNN and Naive Bayes from scratch, benchmarking against SVM, Random Forest and Gradient Boosting.
  • Applied data preprocessing, feature scaling and PCA.
  • Achieved 97.3% accuracy and 0.996 ROC-AUC with an RBF-kernel SVM.
  • Tools: Python, NumPy, Pandas, Scikit-learn.

2. Sentiment Analysis using Deep Learning and NLP.

  • Performed sentiment classification on text data using NLP techniques.
  • Utilized libraries like NLTK, spaCy for preprocessing and Gensim for word embeddings.
  • Built Bi-directional LSTM neural network from scratch using PyTorch.
  • Tools: Genism, PyTorch, spaCy, NLTK.

3. Gastronomic Inference Engine

  • Developed and deployed a full-stack review platform using HTML, CSS, and JS.
  • Integrated with SheetDB and Google Sheets for data management.
  • Trained and deployed ML based NLP Sentiment models (Logistic Regression, SVM, Naive Bayes)
  • Achieved 85% accuracy, hosted on Hugging Face and built Looker Studio dashboard.
  • Tools: HTML, CSS, JS, Python, Scikit-learn, NLTK, Pandas, Matplotlib, Hugging Face, SheetDB.

4. Pneumonia Detection from X-rays using Deep Learning

  • Trained custom CNN and transfer learning models for medical image classification.
  • Achieved best performance with ResNet50 (LR = 0.003), reaching 81.25% validation accuracy and 80.6% macro F1-score and evaluated class-wise performance (Normal: 84.2%, Pneumonia: 76.9%)
  • Applied GradCAM for interpretability.
  • Implemented tuning techniques like GridSearchCV and learning rate scheduling.
  • Tools: Python, PyTorch, ResNet18/50, DenseNet121, EfficientNet-B0, VGG19, Grad-CAM.
